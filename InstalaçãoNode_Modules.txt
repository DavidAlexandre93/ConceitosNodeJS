Instalar Node_Modules

Primeiro isntalar:
npm i loadsh

Instalar:
npm i  -g nodemon

Inicializador de Projetos:
npm init
npm init -y //para responder sim para todas as perguntas

Instalando dependencias e salvando no package.json
npm i --save axios

Instalando dependicias a partir do arquivo json
npm i

Para iniciar teste apartir do nodemon
npm start // como padrao
npm run dev // caso seja algum criado
configurar o arquivo package.json para essa determinada função
